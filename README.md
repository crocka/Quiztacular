LHL Node Skeleton
=========

## Project Setup

Install dependencies with `npm install`.

## Getting Started

1. Run `npm start`
2. Visit `http://localhost:8080/`

## Dependencies

- Node 10.x or above
- NPM 5.x or above
- PG 6.x
- "bcrypt": "^5.0.1",
- "chalk": "^2.4.2",
- "cookie-session": "^2.0.0",
- "dotenv": "^2.0.0",
- "ejs": "^2.6.2",
- "express": "^4.17.1",
- "morgan": "^1.9.1",
- "pg": "^8.5.0",
- "sass": "^1.35.1"

## Running Webpack Development Server

```sh
npm start
```
